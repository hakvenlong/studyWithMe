Pomodoro + Weather App
Simple, beautiful and minimal Pomodoro Timer + Weather Dashboard built with React + Vite.
Live Demo: https://study-with-me-git-main-hak-venlongs-projects.vercel.app/
Features

Pomodoro Timer (25 min work / 5 min short break / 15 min long break)
Fixed calm background (no switching distraction)
Auto weather detection using your location (fallback to Berlin)
Glassmorphism UI
Fully responsive
API key stored safely in .env

Tech Stack

React 18 + Vite
OpenWeatherMap API (forecast)
Geolocation API
React Icons
Pure CSS (glassmorphism)
